$ git status: kiểm tra trạng thái dữ liệu
$ git add . :lưu tất cả file
$ git commit -m "(tiêu đề)": commit file
$ git remote add (tên file) (Địa chỉ repository.git): remote file lên repository
$ git push (tên file): push file
$ git push --set-upstream (tên file) master: upload dữ liệu lên github
$ git log: xem thời gian và (ID)

$ git clone (Địa chỉ repository.git): lấy 1 file dữ liệu về máy tính
$ git pull: upload dữ liệu trên Github về PC
